# Examples
[Files and Directories](https://www.golangprograms.com/files-directories-examples.html)

***

```go
$ go doc os
$ go doc os File
$ go doc os Create
$ go doc os Close

$ go doc -src os Create
$ go doc -src os Close


$ do doc os Stat
$ do doc os FileInfo
$ do doc -src os Stat
$ do doc -src os FileInfo

$ go doc os IsNotExist
$ go doc os MkdirAll

```
