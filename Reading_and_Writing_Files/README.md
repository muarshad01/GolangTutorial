## XML

```go
$ go doc ioutil ReadFile
$ go doc iotuil WriteFile

$ go doc xml
$ go doc xml Unmarshal
$ go doc xml MarshalIndent
```

## JSON

```go
$ go doc ioutil ReadFile
$ go doc iotuil WriteFile

$ go doc json
```
